◆ hn_client_destroy()

void hn_client_destroy ( hn_client *  client)

Closes the connection with the global menu server.

This method closes the connection with the server and destroys the hn_client and all the hn_objects created during the session.
The object_destroy event is called for every hn_object before they are destroyed.

Warning
This method is also called by the library when the connection with the server is lost so it must not be called after a method returns HN_CONNECTION_LOST.
Parameters
client

int hn_client_dispatch_events ( hn_client *  client,
int  timeout 
)

Dispatch events received from the global menu server.

Blocks until a server event is received or timeout is reached and dispatches the events.

Parameters
clientPointer to a valid hn_client struct returned by the hn_client_create() method.
timeoutNumber of milliseconds to block until an event is received. Passing the value -1 disables the timeout.
Returns
HN_SUCESS if events where received or timeout was reached.
HN_CONNECTION_LOST if the connection with the server was lost.
